How to Test Role-Based Navigation
  1. Look at the role dropdown in the header (next to the bell icon)
  2. Click on the dropdown to see available roles
  3. Select a different role to see the navigation change
  4. Notice how the sidebar menu updates based on the selected role
  5. Try accessing different pages to see role-based access control
Note: The navigation menu in the sidebar will automatically update when you change roles using the dropdown in the header.
